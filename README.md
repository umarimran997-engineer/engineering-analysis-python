# Mechanical Engineering Analysis Using Python

This repository contains applied mechanical engineering analysis projects implemented in Python. The analyses demonstrate real-world engineering evaluation techniques including stress analysis, thermal performance modelling, and electric vehicle (EV) energy efficiency studies using physics-based methods and numerical computation.

---

## üöÄ Overview of Projects

### 1. Structural Stress Analysis
- Performs parametric stress evaluation for a component under varying force loads.
- Calculates stress values and compares them with allowable limits.
- Identifies safety margins and decision logic for design evaluation.
- Visualises stress vs force relationships using clear, labelled plots.

### 2. Thermal Parametric Analysis
- Modelling temperature rise for a heated rod under multiple heating scenarios.
- Implements safety factor calculations and design margin assessments.
- Provides decision logic to assess thermal safety across scenarios.
- Produces visual comparisons of temperature profiles.

### 3. EV Energy Consumption & Efficiency Analysis
- Models energy consumption per kilometre for an electric micro-vehicle.
- Includes rolling resistance and aerodynamic drag to compute power demands.
- Performs speed sweeps and optimisation analysis to determine most efficient speed.
- Estimates vehicle range based on battery capacity and consumption values.
- Decision checks at real driving speeds to assess range feasibility.

---

## üß† Key Features

- **Physics-based modelling:** Uses real mechanical engineering formulas for force, stress, drag and energy.
- **Parametric analysis:** Evaluates multiple scenarios (e.g., loads, heating rates, speeds) to identify optimal conditions.
- **Decision logic:** Automates engineering checks (e.g., safety status, allowable limits).
- **Visualisation:** Uses Matplotlib for clear, publication-quality plots.
